# autoclicker

An autoclicker for all your autoclicking needs.

# documentation
General

`r <number of times>` - repeats the profile for a number of times (use n for infinity)

`# <text>` - lines that start with a pound sign are ignored

Mouse

`m` - clicks the mouse

Keyboard

`p <key code>` - presses the key with the specified key code (does not release)

`r <key code>` - releases the key with the specified key code

`t <key code>` - presses & releases the key with the specified key code
